# 工具

## 电脑配置

### 显示文件名与隐藏文件

```
# 显示隐藏文件
任一文件夹=>查看=>勾选隐藏的项目
# 显示文件扩展名
任一文件夹=>查看=>选项=>查看=>取消勾选 隐藏已知文件类型的扩展名
```
